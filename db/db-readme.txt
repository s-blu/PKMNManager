DB Readme zum Pkmmanager

Die Datenbank, die genutzt wird, ist pkmmanager.db - dort liegen eure Daten drin.

Solltet ihr die Datenbank KOMPLETT! resetten wollen:
- benennt pkmmanager.db um - zum Beispiel in pkmmanager-alt.db 
- kopiert pkmmanager-empty.db (anwählen, STRG+C, abwählen, STRG+V) 
- Nennt die kopierte nun in pkmmanager.db um und ihr habt eine leere Datenbank.

Alternativ ändert einfach den Pfad im Quellcode.
